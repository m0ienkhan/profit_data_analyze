Sales & Profit Analysis Project

Overview : This project analyzes sales and profit data to extract meaningful business insights. By combining multiple datasets and performing exploratory data analysis (EDA), the project identifies trends, patterns, and performance differences across categories, sub-categories, and regions.
The goal is to transform raw transactional data into actionable insights that can support better business decision-making.

Objectives : 
Analyze sales and profit distribution across categories and sub-categories
Identify high-performing and low-performing products
Evaluate regional (state-wise) performance
Understand the relationship between sales quantity and profit
Provide insights for improving profitability

Dataset Description

The project uses two datasets:

1. Orders Dataset

Contains order-level information:

Order ID
State
Category
2. Details Dataset

Contains product-level details:

Order ID
Sub-Category
Quantity
Profit

Both datasets are merged using Order ID to create a unified dataset for analysis.
